# 贡献指南

[English](./CONTRIBUTING_en.md) | 中文

---

感谢你参与建设 **ozw**！这个项目旨在打造一个纯粹的 AI 编程工作台，所以我们希望代码保持简洁、可测试且稳定。

### 在你开始之前

- **先开 Issue：** 如果你打算做一个大改动或重构，请先开个 Issue 大家一起讨论。
- **小步快跑：** 尽量提交小的 PR，这样评审和合并都会快很多。
- **兼容性：** 确保你的改动不会损坏现有的用户数据或工作流状态。

### 开发环境搭建

```sh
pnpm install
pnpm dev
```

你需要准备好 Node 22、pnpm 10.33+ 以及 `oz` 工具。

### 关于测试

我们非常看重测试！请编写能够描述**真实用户行为**的测试，而不仅仅是检查某个按钮是否存在。

- `pnpm typecheck` - 每次改完代码都跑一下。
- `pnpm test:server` - 涉及后端逻辑时必跑。
- `pnpm test:e2e` - 涉及 UI 或完整流程改动时必跑。

---

## 💡 代码风格

- 遵循现有的 TypeScript 和 React 模式。
- 保持简单，不要过度抽象。
- 只在解释“为什么这么做”（而非“做了什么”）时添加注释。
